# ikforth Build Instructions

## Pre-requisites

* SConstruct 2.4
* flat assembler version 1.71.51
  * `listing` utility built from sources provided with flat assembler package
* GCC (Linux)
* mingw32 (Linux/Windows)
  * symbolic link `mingw32-g++` -> `/usr/bin/i686-w64-mingw32-g++`
* wine 32 bits (Linux)
