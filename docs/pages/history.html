History of development

5 Jul 1999
  1. Hi again! I wrote last record long time ago ( half of a year! )... I have to work on my diploma all this time - that's why there were no updates and/or improvements to Callisto for Java and accompanied packages. But soon I will release new Callisto for Java, rewritten from the very begining using JDK 1.2.
24 Dec 1998
  1. Class jdev.io.FilenameFilterX removed because of class javax.swing.filechooser.FileFilter existance.
7 Dec 1998
  1. Some bugs fixed in Callisto for Java 0.800.x.x editor ( jdevo.callisto.* packages ).
20 Nov 1998
  1. Callisto for Java packages ( jdev.callisto.* ) renamed to jdevo.callisto.*. Callisto for Java will be rewritten from the very begining as soon as I receive the new release of Java ( JDK 1.2 final ). This new version will be released under jdev.callisto.* packages.
30 Oct 1998
  1. New methods added to jdev.plugin.Plugin: getJMenu, getRoot.
  2. Methods setManager and releaseManager was public in class jdev.plugin.Plugin. They have package access rights now for security reasons.
  3. Classes jdev.awt.resource.Buttons and jdev.awt.resource.Hints removed. Property files ( jdev.awt.resource.Hints.properties and jdev.awt.resource.Buttons.properties ) used now.
27 Oct 1998
  1. jdev.io.Chainer bug fixed.
  2. Packages jdev.io.fs and jdev.io.filter removed. All classes from these packages moved to jdev.io package.
18 Oct 1998
  1. Third release of Callisto for Java ( version 0.800.3.x Beta ) is available for download.
9 Oct 1998
  1. Moving to Swing finished.
  2. Classes that have Swing equivalents moved from jdev.awt.* to jdev.libx.awt.*.
  3. Packages jdev.libx.* renamed to jdevx.*.
7 Oct 1998
  1. Some new functionality added to jdev.lang.VersionControl.
4 Oct 1998
  1. Licensing scheme changed. Using GNU General Public License and GNU Library General Public License.
24 Sep 1998
  1. Started to move Callisto from using AWT and custom written components to Swing. Swing 1.0.3 used.
21 Sep 1998
  1. Package jdev.awt.outline renamed to jdev.awt.tree.
  2. Class jdev.awt.tree.Outline renamed to jdev.awt.tree.TreeView.
8 Sep 1998
  1. Class jdev.compiler.CompilerOutputParser removed.
26 Aug 1998
  1. All packages in jdev.fc.* renamed to jdev.*.
  2. Setup bundle splitted into several parts, which can be downloaded independently.
18 Aug 1998
  1. Second release of Callisto for Java ( version 0.800.2.x Beta ) is available for download.
16 Aug 1998
  1. A large piece of work done to jdev.awt.outline.Outline.
15 Aug 1998
  1. Callisto manuals updated.
  2. jdev.plugin.FileObserver changed.
14 Aug 1998
  1. Classes jdev.plugin.DesktopObserver and jdev.plugin.DesktopOwner was removed. Classes jdev.plugin.FileObserver and jdev.plugin.FileOwner contain old functionality now.
9 Aug 1998
  1. Version information representation changed. See jdev.lang.VersionControl.
5 Aug 1998
  1. Callisto manuals updated.
  2. Classes from package jdev.awt.edit was moved to jdev.awt.editor.
    Package jdev.awt.edit removed.

  1. Added support for syntax highlighting to jdev.awt.editor.EditorView.
    See jdev.awt.scanner package.
1 Aug 1998
  1. New classes in jdev.io package. See jdev.io.ProgressObserver, jdev.io.ProgressReader, jdev.io.ProgressWriter, jdev.io.ProgressInputStream, jdev.io.ProgressOutputStream.
30 Jul 1998
  1. Added support for custom keyboard layouts to jdev.awt.editor.EditorView.
    See jdev.awt.editor.KeyDecoder
  2. Constants from class jdev.awt.MenuKey was moved to jdev.awt.AWTUtils.
    Class jdev.awt.MenuKey removed.
23 Jul 1998
  1. Methods from class jdev.awt.FocusRect was moved to jdev.awt.AWTUtils.
    Class jdev.awt.FocusRect removed.
  2. Methods from class jdev.awt.MenuUtil was moved to jdev.awt.AWTUtils.
    Class jdev.awt.MenuUtil removed.
25 Jun 1998
  1. First release of Callisto for Java ( version 0.790.1.x Beta ) is available for download.
24 Jun 1998
  1. Added support for selection, cut/copy/paste/clear operations ( using system clipboard ) to jdev.awt.editor.EditorView.
16 Jun 1998
  1. History page started
  2. All packages in jdev.fc.awt.* renamed to jdev.awt.*.
